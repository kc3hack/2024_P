## プロダクト説明
関西らしさを発見することで、東京駅の地下通路、「八重洲出口」からの脱出を図る3D脱出ゲームです。
8番出口(https://store.steampowered.com/app/2653790/_/?l=japanese)をオマージュしています。
プレイヤーは、東京なのに「関西っぽい」ものを見つけたら、即座に引き返します。もし引き返さなければ、ループする地下通路から脱出することは一生できません。
